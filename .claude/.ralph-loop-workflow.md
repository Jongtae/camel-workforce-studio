# Ralph Loop Workflow: Context → Pipeline → Issue

## Task
Run the complete studio workflow cycle:
1. Build context from AI-Fashion-Forum repo
2. Run commitment + workforce debate (1 round)
3. Create GitHub issue bundle
4. Update run ledger and metadata

## Commands to Execute

### Step 1: Activate environment and build context
```bash
source .venv/bin/activate && python scripts/context-builder/build_context.py \
  --repo Jongtae/AI-Fashion-Forum \
  --source-dir /Users/jongtaelee/Documents/AI-Fashion-Forum
```

### Step 2: Run pipeline with issue creation
```bash
python scripts/pipeline/run_studio.py \
  --repo Jongtae/AI-Fashion-Forum \
  --source-dir /Users/jongtaelee/Documents/AI-Fashion-Forum \
  --rounds 1 \
  --create-issue \
  --issue-type bundle \
  --epic-label epic:forum-actions \
  --issue-milestone "Sprint 1 - Identity Loop Vertical Slice" \
  --task-assignee jongtae \
  --task-assignee alice \
  --with-sprint
```

## Success Criteria
- Context files generated in `context/`
- Pipeline output in `scripts/requirement-debate/outputs/`
- GitHub issue created successfully
- Run ledger updated in `context/history/run-ledger.jsonl`

## Completion Signal
When all steps are complete and issues are created:
```
<promise>WORKFLOW COMPLETE</promise>
```
